# Quickstart — валидация TimeFocus MVP

## Предварительные условия

- Flutter stable (Dart SDK ^3.12), Xcode/Android SDK.
- Рабочий каталог: `app/`.

## Сборка и запуск

```bash
cd app
flutter pub get
dart run build_runner build --delete-conflicting-outputs   # freezed/injectable/drift/gen
flutter analyze                                            # чисто
flutter test                                               # все зелёные
flutter run                                                # устройство/эмулятор
```

## Сквозные сценарии проверки (соответствуют User Stories спецификации)

1. **Трекер (US1)**: тап по «Работа» в сетке → карточка с тикающим таймером; пауза →
   карточка вниз; свернуть приложение на 2+ мин → таймер корректен (SC-002); стоп →
   запись в Истории.
2. **Помодоро (US2)**: у «Работы» тип normal + перерыв «Перерыв»; старт → через 25 мин
   уведомление pomodoroFinished; тап → перерыв стартует, Помодоро completed. Запуск
   «Туалет» во время работы → Помодоро interrupted + тост. Пауза Помодоро-активности →
   интервал interrupted (clarification).
3. **Вода/HUD (US3)**: тап стакана → +200 мл, полоска обновилась; удержание → список
   напитков; interval-режим → уведомление через N мин после питья; при активном Помодоро
   стакан мигает, уведомление ждёт перерыва; иконка туалета по приоритету.
4. **Расписание (US4)**: строгий обед 13:00; старт Помодоро 12:45 (25 мин) → немедленное
   предупреждение; в 13:00 Помодоро прерван, уведомление пришло, активность стартует по тапу.
5. **Cold start (US5)**: убить приложение при работающем Помодоро → дождаться уведомления →
   тап → приложение открылось и выполнило сценарий (SC-003).
6. **История (US6)**: режимы/периоды переключаются, шапка (время без «Сна», Помодоро ✓/✗,
   вода) пересчитывается; правка интервала с пересечением → предупреждение, но сохраняется;
   конец < начала → блокировка.
7. **Настройки (US7)**: смена темы/языка — мгновенно (SC-008); создание активности → в сетке;
   системную нельзя удалить.
8. **Онбординг (US8)**: первый запуск → онбординг, skip → трекер с 12 системными
   активностями; повторный запуск — онбординга нет.
